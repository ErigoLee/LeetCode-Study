## Palindrome Number
A `Palindrome` number means that when the number is read backward, it is then same as when the number is read forward.


## Algorithms

```python
x_str = str(x)
x_len_half = int(len(x_str)/2)
```
Then, I use a for loop to check whether the value at index `i` is the same as the value at index `x_len - 1 - i`. </br>
(I use a for `loop` to check if characters at symmetric positions are the same.) </br>
